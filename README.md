## Overview

crypto is a browser-first cryptography research project organized around interactive HTML notebooks and hands-on experimental material. It covers RSA, Diffie-Hellman, ECC, and the discrete logarithm problem in a format that can be opened, reviewed, and expanded directly in a web browser.

In addition to the core cryptography content, the repository includes TRNG audit tools, AI chat research logs, and a p2p protocol layer. That combination makes it suitable for security study, structured testing, and exploratory work. The project also references augmented-reality, text-driven RPG material, which gives it a wider interactive scope than a conventional lab notebook.

## How to Use

Treat the repository as a web-based research environment for reading notebooks, checking experiments, and moving through the available content.

Typical flow:
1. Open the main interface in a web browser.
2. Pick a topic such as RSA, ECC, or TRNG auditing.
3. Read the notebook or log entries associated with that topic.
4. Follow linked research artifacts, p2p notes, or adventure content.
5. Add your own experiments or observations by extending the HTML pages.

For research sessions, keep your notes close to the experiment being tested so results are simpler to compare afterward.
